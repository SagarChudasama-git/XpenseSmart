# About

XpenseSmart is a powerful and intuitive expense tracking solution designed to help individuals and businesses manage their finances effortlessly. With seamless integration between a web application and desktop software.

# Features

-  Users can input expense details (amount, category, date, description).
- Modify or remove incorrect entries.
- Display a list of past transactions with filters (date range, category, etc.).
- Users can define custom expense categories.
- Use JavaScript libraries (like Chart.js) to show expense trends.
